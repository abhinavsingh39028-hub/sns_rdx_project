# ⚜️ Samajik Nyay Sangh (SNS) - National Constitutional Portal

A production-grade, highly-structured React web application designed to host, query, and compare the official model charter of the Samajik Nyay Sangh against the Constitution of India framework.

## 🚀 Key Architectural Features
- **Verbatim Constitutional Reader:** Complete untruncated text mapping of all Parts, Articles, and Clauses.
- **Side-by-Side Comparison Matrix:** Real-time dual-column interface mapping structural proposals alongside the Constitution of India.
- **Keyword Action Router Engine:** High-performance local search index with auto-scrolling and highlighting UI mechanics.
- **Induction Oath & Gated Voting Modules:** Fully functional client-side mock-ups for verification and decentralized compliance checks.

## 💻 Tech Stack
- **Frontend Framework:** React.js (Vite Engine)
- **Styling Architecture:** Tailwind CSS (Legal Serif Typographic System)
- **State Management:** React Hooks (`useMemo`, `useRef`, `useEffect`)
